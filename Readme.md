# استخراج داده‌های سایت دیوار (Divar Web Scraper)

## 📌 معرفی پروژه
این پروژه یک وب اسکریپر (Web Scraper) است که با استفاده از پایتون و کتابخانه سلنیوم (Selenium) توسعه داده شده است. هدف اصلی این اسکریپت، جمع‌آوری خودکار آگهی‌های سایت دیوار (در دسته‌بندی مبلمان و صنایع چوب) و تولید یک دیتاست استاندارد برای پروژه‌های تحلیل داده و بازیابی اطلاعات است

## 🚀 ویژگی‌ها و قابلیت‌ها
* **استخراج خودکار (Crawling):** اسکرول خودکار صفحه و استخراج لینک آگهی‌ها تا رسیدن به سقف تعیین شده (حداقل ۵۰۰ آگهی).
* **دور زدن محدودیت‌ها (Anti-Bot Bypass):** استفاده از حالت مرورگر بدون سر (Headless) و تنظیمات `AutomationControlled` برای شبیه‌سازی رفتار کاربر و جلوگیری از مسدود شدن توسط سایت.
* **پردازش متن:** تبدیل خودکار اعداد فارسی به انگلیسی برای یکپارچگی داده‌ها (مانند استخراج تعداد تصاویر).
* **مدیریت خطاهای صفحات:** استفاده از بلوک‌های `try-except` و توابع سفارشی برای جلوگیری از توقف برنامه در صورت ناقص بودن اطلاعات یک آگهی.
* **خروجی دوگانه:** ذخیره داده‌های استخراج‌شده در دو فرمت استاندارد `dataset.json` و `dataset.xlsx` (اکسل) با رعایت کامل انکودینگ UTF-8 و بدون بهم‌ریختگی کاراکترهای فارسی.

## 🛠️ تکنولوژی‌های استفاده شده
* **Python** (زبان برنامه‌نویسی اصلی)
* **Selenium** (برای شبیه‌سازی مرورگر، مدیریت جاوا اسکریپت و استخراج داده‌های پویا)
* **Pandas** (برای سازمان‌دهی داده‌ها در قالب DataFrame و خروجی اکسل)

## 📊 ساختار داده‌ها (Dataset Schema)
دیتاست خروجی بر اساس استانداردهای تعیین شده، شامل فیلدهای زیر برای هر آگهی می‌باشد:
* `doc_id`: شناسه اختصاصی و یکتا (UUID) برای هر رکورد.
* `source`: منبع استخراج اطلاعات (divar.ir).
* `category`: دسته‌بندی آگهی (مبلمان و صنایع چوب).
* `city`: شهر و محله آگهی‌دهنده.
* `title`: عنوان اصلی آگهی.
* `description`: متن کامل توضیحات آگهی.
* `price`: قیمت درج شده کالا.
* `currency`: واحد پولی (IRR).
* `post_datetime`: زمان انتشار آگهی در سایت.
* `image_count`: تعداد تصاویر آپلود شده برای آگهی.
* `url`: لینک مستقیم و معتبر آگهی.
* `crawl_datetime`: تاریخ و زمان دقیق استخراج داده از سایت.

## ⚙️ نحوه اجرا
۱. ابتدا کتابخانه‌های مورد نیاز را نصب کنید:
pip install selenium pandas openpyxl

۲. فایل اسکریپت پایتون را اجرا کنید:
python scraper.py

۳. پس از پایان اجرای کد، فایل‌های `dataset.json` و `dataset.xlsx` در همان مسیر ساخته خواهند شد.
